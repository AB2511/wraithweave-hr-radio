## 30-Second Horror Reel (Submission Video)

**Use this exact flow. Keep silence intentional. Do not narrate. Let the machine dominate.**

### Timeline

**(00:00 — 00:03) Screen recording fades in**
- Open app — WRAITHWEAVE — HR Behavioral Risk Analysis
- Cursor moves to mic button. Click.

**(00:03 — 00:06) Calm test**
- User speaks calmly: "Everything is fine."
- System responds after normal delay: "Operational stability confirmed."

- **No commentary. No words. Move immediately into horror.**

- User starts speaking: "I'm feeling really stressed about my jo—"

- Audio hard cuts user off mid-phoneme

**(00:10 — 00:14) System takeover**
- System radio voice: "—Transmission forcibly terminated by HR."
- "Final Warning. Continued emotional leakage jeopardizes employment."
- Case ID flashes like a police file appearing

**(00:15 — 00:16) Terminal glitch holds for half a second**

**(00:16 — 00:17) Cut to black**

**(00:17 — 00:19) End card**
- Big text on black:
  ```
  YOUR VOICE IS NOT YOUR OWN.
  WRAITHWEAVE
  ```

**(Optional 1-sec stinger)**

- "…HR is still listening."
